如何搭建一个属于自己的hexo博客？

发表于 2020-04-30 更新于 2020-04-13 分类于 建站

如何在windows环境下搭建hexo博客及常见问题:

准备工作：

1.首先下载安装 Node.js(LTS)

2.安装 Git Bash

(1)出现的问题：git下载太慢了；
解决方法：F12找到下载链接，利用迅雷下载;

(2)安装教程可参考cnds： 点击前往

一、安装hexo

首先运行git;安装hexo框架；输入

利用hexo -v检测是否安装成功

出现这个就表示安装成功了

二、生成博客文件

接下来建立一个存放博客的文件夹；比如我放在d盘；建立一个文件夹
在git输入:

hexo init

注意路径：D：/bolg
如图:

创建成功后在浏览器中输入
localhost：4000
就会出现如图所示:

在
D:\blog\source_posts
路径下建立一个.md文件，就可以开始写你的博客了

三、远程部署

要使用远程部署需要先安装hexo-deployer-git

接着在github建立一个仓库
注意：仓库名一定要是你github的名字
例如xxxx.github.io

接着改_config.yml文件

把最后几行进行更改

最后输入命令

hexo clean
hexo g
hexo d

遇到的问题：在hexo d部署时，每次都要输入账号密码
解决方法：使用ssh-key生成秘钥

在浏览器中输入你的仓库名，就可以看到你的博客啦~
